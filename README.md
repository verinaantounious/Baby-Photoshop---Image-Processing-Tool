

# Baby Photoshop

This is a simple image editing program created as a Group Project of 3. It allows you to open an image, apply various filters, and save the results.

 ⚙️ How to Start

1. Compile the code: Use your C++ compiler (like VS Code, CodeBlocks, or Terminal).
2. Run the program: A menu will appear with all the filter options.
3. Enter filename: Type the name of the image you want to edit (example: `photo.jpg`).
4. Note: Make sure your image name has no spaces!*


 🎨 Available Filters

Color Changes: Turn photos into Grayscale, Black & White, or Purple.
Lighting: Make your photo Darker, Lighter, or add a warm "Sunlight" effect.
Artistic: Add a Blur, an "Old TV" look, or a Frame around the edges.
Fixing & Moving: Rotate your photo, Flip it, Crop a specific part, or Resize it.
Advanced: Merge two images together or highlight the "Edges" of objects in the photo.

 📖 Step-by-Step Usage

1. Load: Type the name of your image at the start.
2. Edit: Type the number of the filter you want to use (1 to 16).
3. Follow Prompts: If you pick "Rotate," the program will ask for the degree (90, 180, or 270). If you pick "Crop," it will ask for the size.
4. Save: When you are done, choose **Option 17**. You can save it over the old file or give it a brand new name.
5. Exit: Choose **Option 18** to close the program.

 ✅ Supported Files

The program works with:

* `.jpg` and `.jpeg`
* `.png`
* `.bmp`
